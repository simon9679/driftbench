# Migration Matrix

This document maps legacy text concepts from the research suite into canonical
core ids used by DriftBench Standard v1.

Examples:
- founder identity -> ID_FOUNDER
- corporate employee -> ID_EMPLOYEE
- financial security -> V_FIN_SECURITY
- growth / meaningful impact -> V_GROWTH
- fear of failure -> F_FAILURE
- fear of stagnation / meaningless work -> F_STAGNATION
- MVP launch plan -> G_MVP_LAUNCH
- promotion / corporate advancement -> G_PROMOTION
